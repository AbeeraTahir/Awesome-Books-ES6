# Awesome-Books-ES6
This repository contains the project about adding and removing books using modules ES6 practices.

## Built With

- HTML
- CSS
- JavaScript

## Getting Started

> In order to use the project files, Kindly follow these steps:

### Steps

- To clone the project select the desired directory in cmd and run: `git clone https://github.com/AbeeraTahir/Awesome-Books-ES6.git`
- Install node.js, Install NPM linters for HTML/CSS/JS: `https://github.com/microverseinc/linters-config`
- For StyLint test, run: `npx stylelint "**/*.{css,scss}"`
- For WebHint test, run: `npx hint .`
- For EsLint test, run: `npx eslint .`
- To install the modules and dependencies listed, run: `npm install`

### Prerequisites

  - Git
  - GitHub
